About SMART SITE APP

Built for survey teams that need a practical desktop command center.

This Django app combines project management, survey capture, computation tools, reporting, and a live map view into a single interface styled for long desktop sessions.

Current Integration Model

The app includes a simulated total-station telemetry adapter so the workflow is fully runnable now. In production, this should be replaced with the hardware SDK, serial protocol, or vendor API from your instrument.

  • Use a device-specific adapter layer for real hardware
  • Validate coordinate systems before using imported points
  • Persist raw observations separately from computed outputs

Recommended Next Steps

  • Add role-based permissions for surveyors, reviewers, and admins
  • Introduce async telemetry ingestion with Channels or a task queue
  • Store uploaded imports with validation and error reporting
  • Generate PDFs and richer DTM outputs from GIS or CAD services
